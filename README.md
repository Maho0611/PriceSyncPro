## ✨ 核心优势

- **🌐 官方价格来源**：价格数据直接来自 [OpenRouter](https://openrouter.ai) 的官方定价接口，联网实时获取（12 小时缓存），离线时自动回退到本地快照。
- **🔍 模糊匹配**：自动清理渠道模型名中的前缀/后缀干扰，与 OpenRouter 官方价格表智能匹配。
- **✅ 按需勾选**：匹配结果以表格形式展示，可自由勾选/取消勾选要同步的模型，未匹配的模型单独列出且不会被同步。
- **🔒 绝不改名**：插件只读取渠道已有的模型名，只写入 `ModelRatio`/`CompletionRatio` 价格字段，永远不会修改渠道的模型列表或模型名称。
- **🔄 批量更新**：一键批量更新所有渠道，自动为每个渠道匹配并同步所有命中官方价格的模型。
- **更安全**：通过浏览器 Cookie 进行认证，无需暴露和管理 API Token。
- **更便捷**：直接在 New API 后台页面使用，无需切换任何外部工具。

## 🚀 快速开始指南

### 第一步：登录 New API 后台

- 打开你的 New API 管理后台并成功登录。
- **重要**: 插件需要使用浏览器的登录状态 (Cookie) 来进行安全认证。

### 第二步：打开侧边栏，选择渠道

在 New API 后台的**任意页面**，点击浏览器工具栏的 **PriceSyncPro** 图标打开侧边栏。

1.  在“目标渠道”下拉框中选择一个**已存在**的渠道。
2.  插件会自动读取该渠道当前已配置的模型列表，并与 OpenRouter 官方价格进行模糊匹配。
3.  匹配结果会显示在表格中：
    *   **已匹配**的模型默认勾选，可查看匹配到的官方模型名及输入/输出价格。
    *   **未匹配**的模型会单独标记，勾选框禁用，不会被同步。
4.  如需排除某些模型，取消对应勾选即可；也可用表头勾选框一键全选/全不选。
5.  点击“**同步选中价格**”按钮，插件只会更新被勾选模型的 `ModelRatio`/`CompletionRatio`，其余现有配置（包括其他渠道、其他模型）不受影响，且**不会**修改任何模型的名称。

### 🔄 批量更新所有渠道

如果你想一次性为所有渠道同步价格：

1.  点击“**批量更新所有渠道**”按钮。
2.  确认操作后，插件会自动对每个渠道执行：读取当前模型列表 → 匹配官方价格 → 同步所有匹配成功的模型价格。
3.  完成后会展示完整的结果报告，包括每个渠道的成功/失败/跳过（无匹配模型）情况。

## 🎯 核心功能详解

### 🌐 官方价格数据源
- 参考价格数据来源 [OpenRouter](https://openrouter.ai)，联网实时获取（12 小时缓存），离线或接口异常时自动回退到本地快照。
- 快照文件 `official_prices.json` 可通过 `node scripts/update-official-prices.js` 重新生成。

### 🔍 模糊匹配规则
- 自动去除模型名中的供应商前缀（取路径最后一段）。
- 清理描述性后缀/标记（如括号、方括号标签、`-thinking`、中文标注等）。
- 尝试点号/破折号互换、大小写变体等常见命名差异，提升匹配命中率。

### 🔒 只读渠道模型，只写价格
- 插件读取渠道当前的模型列表仅用于匹配价格，**从不**调用任何会修改渠道 `models`/`model_mapping` 字段的接口。
- 同步操作只精确覆盖被勾选模型对应的 `ModelRatio`/`CompletionRatio` key，其余现有配置原样保留。

## ❓ 常见问题 (FAQ)

**Q1: 插件为什么比 API Token 更安全？**
**A:** 插件运行在你的 New API 页面中，直接复用浏览器的登录状态 (Session Cookie) 进行认证，全程无需在插件中输入或存储你的 API Token，避免了 Token 泄露的风险。

**Q2: 会覆盖我其他渠道或其他模型的配置吗？**
**A:** 完全不会。同步操作只精确覆盖你勾选的那些模型对应的价格 key，其余所有现有配置（其他渠道、其他模型）均原样保留。

**Q3: 会修改我的模型名称吗？**
**A:** 不会。插件只读取渠道已有的模型名用于匹配价格，从不调用修改模型列表/名称的接口。

**Q4: 点击插件图标没反应或提示认证失败怎么办？**
**A:** 这通常是因为登录状态已过期。请在 New API 后台页面**刷新一下**，如果需要的话重新登录，然后再打开插件即可。

**Q5: 某个模型显示"未匹配"怎么办？**
**A:** 说明该模型名在 OpenRouter 官方价格表中没有找到对应条目（可能是模型名称差异过大，或该模型不在 OpenRouter 上架）。这类模型不会被同步，你可以手动在 New API 后台设置其价格。
